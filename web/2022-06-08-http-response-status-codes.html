304

4xx ​

表示请求可能出错，妨碍了服务器的处理

sh
# (错误请求)服务器不理解请求的语法
400
# 表示发送的请求需要有通过HTTP认证的认证信息
401
# (禁止)服务器拒绝请求
403
# (未找到)服务器找不到请求地址
404

sh
# (服务器内部错误)服务器遇到错误，无法完成请求
500
# 表示服务器处于停机维护或超负载，无法处理请求
